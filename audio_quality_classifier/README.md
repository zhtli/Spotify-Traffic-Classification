## Online quality (traffic-based)

Filters incoming audio packets and estimates the effective bitrate.  
Classifies audio quality based on measured bitrate.

---

## Quality levels

- `low`
- `normal`
- `high`
- `high_premium`
- `very_high`

---

## Output

JSON file containing:

- Capture metadata
- Estimated bitrate
- Estimated audio quality
- Active Spotify IPs
- Per-packet traffic data
